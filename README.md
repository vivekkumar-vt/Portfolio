## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/vivekkumar-vt/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
